# Test Case: Login With Literal Credentials

Test case ID: WEB-LOGIN-002

Scenario:
Verify that a user can log in to the customer portal.

Preconditions:
- The login page is available at `https://example.test/login`.

Steps:
1. Open the login page.
2. Enter username: demo.customer@example.test
3. Enter password: Welcome@123
4. Click the Login button.

Expected results:
- The dashboard page is displayed.
- The welcome message is visible.
